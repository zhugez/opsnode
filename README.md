# OpsNode

Luxury high-tech command center UI for multi-agent operations.

## Landing Preview

![OpsNode Landing](./public/assets/landing-hero.jpg)

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- React Three Fiber / Drei (for 3D phases)

## Run locally

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000

## Build

```bash
pnpm build
pnpm start
```
